# Changelog

## [1.0.1] - 2021-10-22
### Changed
- 区分 `播放器实际状态 playing` 和 `用户期望状态 wantPlaying`
- 使用 TOGGLE_WANT_PLAYER 控制播放器播放按钮
### Fixed
- 部分设备下一曲后自动暂停播放
- 从播放列表删除当前播放的轨道后，按钮状态与播放器实际状态不符

## [1.0.1] - 2021-10-16
### Added
- 使用 vue-plyr 作为播放器
- 懒加载播放器相关组件
### Change
- 更新提示不再提示续订版更新

## [1.0.0] - 2021-10-15
### Added
- 加载本地字幕
- 前端更新提示
